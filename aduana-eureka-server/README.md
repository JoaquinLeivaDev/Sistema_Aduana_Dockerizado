# Eureka Server - Sistema Aduana

## Descripción
Servidor de descubrimiento de servicios (Service Discovery) centralizado. Permite la monitorización y el registro de todas las instancias de microservicios del sistema.

## Ejecución
### Local
1. Ejecutar `./mvnw.cmd spring-boot:run`
2. Acceder al dashboard en `http://localhost:8761`.
